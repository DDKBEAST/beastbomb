## Usage

Update the packages

* `Termux`

```bash
pkg up -y
```
* `Kali Linux/Parrot OS`

```bash
apt update -y && apt upgrade -y
```

Clone the repository
```bash
git clone https://github.com/King-Nazim/darkhack
```
Go to the darkhack directory
```bash
cd darkhack
```
Run the script
```bash
bash bomb.sh
```
